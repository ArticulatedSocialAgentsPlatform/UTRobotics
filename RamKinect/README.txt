a simple program to get some data from the kinect and put it on the Apollo network for use in other modules of the UT Robotics setup

This project was created with Microsoft Visual Studio 2015 Enterprise edition.

--- Build notes ---
To be able to build the source, you will need to have the Kinect SDK and Toolkit installed. They are available from:
 - https://www.microsoft.com/en-us/download/details.aspx?id=40278
 - https://www.microsoft.com/en-us/download/details.aspx?id=40276


--- Run notes ---
Not tested, but if the program does not run on a default Windows machine, install the Kinect Runtime from:
 - https://www.microsoft.com/en-us/download/details.aspx?id=40277
